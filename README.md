# Student_Gap-Analysis

Developed a data-driven analytics solution to identify and analyze student learning gaps across grades, books, and chapters. The system retrieves assessment data from MongoDB, processes student performance metrics using Pandas, and classifies learning gaps into categories such as Concept Gap, Execution Issue, Mixed Gap, and Not Attempted. Interactive Jupyter widgets enable educators to filter and explore data efficiently, helping them design targeted interventions and personalized learning plans.

# Features
. Student performance analysis at multiple levels
. Automated learning gap classification
. Weak skill identification
. Interactive dashboard using ipywidgets
. MongoDB integration for scalable data retrieval
. Actionable insights for personalized learning
# Tech Stack
. Python
. Pandas
. MongoDB (PyMongo)
. Jupyter Notebook
. ipywidgets
. dotenv
